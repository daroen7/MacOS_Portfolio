
## 💼 Mac OS Portfolio

A creative and interactive personal portfolio designed with a **macOS-inspired** interface. Built with **React** and styled with **TailwindCSS**, featuring smooth animations powered by **GSAP**. The portfolio showcases my projects, skills, and background in a unique and memorable way.

### ✨ Features
- 🖥️ macOS-inspired UI design
- 🎨 Smooth animations with GSAP
- 📱 Responsive layout with TailwindCSS
- 💼 Projects & skills showcase
- 📬 Contact section

### 🛠️ Tech Stack
- React
- TailwindCSS
- GSAP
